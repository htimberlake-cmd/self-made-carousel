# Portfolio Video Carousel

A lightweight standalone HTML video carousel created for use within a personal portfolio and embedded in Google Sites.

## Purpose

This project is used to present selected video work in a compact, responsive carousel format, with clickable thumbnails that open a larger player overlay.

## Features

- Responsive layout for desktop, tablet, and mobile
- Clickable video thumbnails
- Modal video player
- Previous / next navigation
- Support for YouTube, Vimeo, and direct video links
- Designed for embedding in Google Sites

## Files

- `index.html` — main standalone carousel file
- Additional `.html` files may be added for different categories of work
